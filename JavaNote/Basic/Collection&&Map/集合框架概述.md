# 集合框架概述
## 介绍
java 中的集合框架大致可分为 `Set` `List`  `Queue`  `Map` 这四种,其中:
- Set 代表无序,不可重复的集合;
- List 代表有序,重复的集合;
- Map 代表具有映射关系的集合;
- Queue  是 JDK5 之后添加的,作为一种队列的实现存在。
